## 🚀 Proje Kapsamı ve Yol Haritası
Eğitim boyunca geliştirilecek ve bu repository altında modüler olarak yer alacak tüm konu başlıkları aşağıda listelenmiştir:

- **Flutter 101 (Temel Nesne ve Arayüz Yönetimi):** Core widget mimarileri, layout tasarımları, Stateless/Stateful yaşam döngüleri ve sayfa navigasyon (`Navigator`) yönetimleri.
- **Flutter 202 (Servis ve Veri Yönetimi):** HTTP/Network entegrasyonları (Dio, Retrofit), yerel veri saklama mekanizmaları (Shared Preferences, Secure Storage, Hive), gelişmiş lifecycle takipleri (`didUpdateWidget` vb.) ve hata yakalama senaryoları.
- **Flutter 303 & Mimari (İleri Seviye):** State Management çözümleri (Bloc, Provider, MobX), Clean Architecture mimari yaklaşımları, bağımlılık yönetimi (GetIt), kod üretimi (Code Generation - Freezed, JsonSerializable), Custom Theme ve Localization (Çoklu Dil) altyapıları.
- **Test ve Kalite Süreçleri:** İş mantıklarının doğrulanması adına Unit, Widget ve Integration test senaryolarının koda entegre edilmesi.

## 🛠️ Teknik Altyapı

- **Geliştirme Ortamı:** VS Code
